# Personal Portfolio Website

## Overview

This project was created as a central place to present my work, technical skills, and professional journey. Rather than serving as a simple online résumé, the portfolio was designed to provide a structured and engaging experience where visitors can explore selected projects, learn about my background, and get in touch through an intuitive interface.

The application was built as a full stack solution using React for the frontend, Django for the backend, and PostgreSQL as the primary database. The architecture was designed with scalability and maintainability in mind, making it easy to expand with additional features over time.

Particular attention was given to responsive design, performance, accessibility, and clean visual presentation to ensure a consistent experience across desktop and mobile devices.
